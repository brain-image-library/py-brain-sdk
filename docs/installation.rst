Installation
============

Requirements
------------

* Python 3.8 or higher
* pip

Install via pip
---------------

Install the latest stable release from PyPI::

    pip install brainimagelibrary

Install a specific version::

    pip install brainimagelibrary==0.0.18

Upgrade an existing installation::

    pip install --upgrade brainimagelibrary

Install from source
-------------------

To install the development version directly from GitHub::

    pip install git+https://github.com/brain-image-library/py-brain-sdk.git

Or clone the repository and install locally::

    git clone https://github.com/brain-image-library/py-brain-sdk.git
    cd py-brain-sdk
    pip install .

Verify installation
-------------------

Confirm the package is installed correctly::

    python -c "import brainimagelibrary; print(brainimagelibrary.__version__)"
